# PriorAuth CoPilot

Turn insurance denials into approved appeals, instantly.

## Setup

1. Clone this repository
2. Run `npm install`
3. Create a `.env` file with your Anthropic API key:
   ```
   VITE_ANTHROPIC_API_KEY=your-key-here
   ```
4. Run `npm run dev`

## Deploy to Vercel

1. Push to GitHub
2. Import in Vercel
3. Add environment variable: `VITE_ANTHROPIC_API_KEY`
4. Deploy!
